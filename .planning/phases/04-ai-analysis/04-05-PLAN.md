Create backend/app/services/sentiment_service.py implementing:

1. VADER initialization:
```python
from nltk.sentiment import SentimentIntensityAnalyzer
import nltk

# Download VADER lexicon on first import (idempotent)
try:
    nltk.data.find('sentiment/vader_lexicon.zip')
except LookupError:
    nltk.download('vader_lexicon', quiet=True)

sia = SentimentIntensityAnalyzer()
```

2. analyze_comment_sentiment(comment_text: str) -> dict:
- Call sia.polarity_scores(comment_text)
- Returns: {"neg": float, "neu": float, "pos": float, "compound": float}
- compound ranges from -1.0 (most negative) to +1.0 (most positive)

3. categorize_sentiment(compound_score: float) -> str:
- If compound >= 0.05: return "positive"
- If compound <= -0.05: return "negative"
- Else: return "neutral"

4. analyze_comment_batch(comments: list[str]) -> dict:
- Analyze list of comment texts
- Return summary: {"positive": count, "neutral": count, "negative": count, "avg_compound": float}
- Handle empty list: return all zeros
