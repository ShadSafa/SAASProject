  <action>
Create backend/app/services/openai_service.py implementing:

1. Pydantic models for structured output:
```python
from pydantic import BaseModel, Field

class ViralAnalysisResult(BaseModel):
    """Complete AI analysis of a viral post."""
    why_viral_summary: str = Field(..., description="2-3 sentence explanation")
    posting_time_score: float = Field(..., ge=0.0, le=100.0)
    hook_strength: float = Field(..., ge=0.0, le=100.0)
    emotional_trigger: str = Field(..., description="joy|awe|anger|surprise|sadness|fear")
    engagement_velocity_score: float = Field(..., ge=0.0, le=100.0)
    save_share_ratio_score: float = Field(..., ge=0.0, le=100.0)
    hashtag_performance: float = Field(..., ge=0.0, le=100.0)
    audience_retention: float = Field(..., ge=0.0, le=100.0)
    confidence_score: float = Field(..., ge=0.0, le=1.0)
```

2. analyze_viral_post() function:
- Accept ViralPost ORM object as input
- Build prompt with caption, hashtags, engagement metrics, creator info, post age
- Call OpenAI API: `client.beta.chat.completions.parse(model="gpt-4o", messages=[...], response_format=ViralAnalysisResult)`
- Return ViralAnalysisResult with all 7 algorithm factors scored 0-100
- Handle API errors (rate limit, auth, timeout) with HTTPException

3. Prompt engineering:
Include in prompt: "Analyze this viral Instagram post and score 7 algorithm factors (0-100 each). Consider: posting time optimal for audience timezone? Hook strength in first 3 seconds/caption opening? Emotional trigger (joy/awe/anger/surprise/sadness/fear)? Engagement velocity (how quickly gained engagement)? Save/share ratio vs likes (high saves = high value)? Hashtag relevance and trending status? Audience retention throughout content?"

Use research example from 04-RESEARCH.md lines 408-482 as reference.
