  <name>Viral Score Calculation</name>
  <files>

  <behavior>
    calculate_viral_score(engagement_count: int, follower_count: int, post_age_hours: float) -> float

    Formula: (engagement_count / follower_count) * velocity_multiplier * 100, capped at 100.0

    Velocity multipliers by post_age_hours:
    - < 1 hour: 3.0 (exceptional speed)
    - 1 to < 2 hours: 2.5 (very fast)
    - 2 to < 4 hours: 2.0 (fast)
    - 4 to < 12 hours: 1.5 (moderate)
    - 12 to < 24 hours: 1.0 (normal)
    - >= 24 hours: 0.5 (slow)

    Edge cases -> expected outputs:
    - engagement=1000, followers=10000, age=1.0 -> (0.1 * 2.5 * 100) = 25.0
    - engagement=1000, followers=10000, age=0.5 -> (0.1 * 3.0 * 100) = 30.0
    - engagement=0, followers=10000, age=1.0 -> 0.0
    - engagement=1000000, followers=1000, age=0.5 -> capped at 100.0
    - engagement=100, followers=0, age=5.0 -> 0.0 (not division error)
    - engagement=100, followers=1000, age=24.0 -> (0.1 * 1.0 * 100) = 10.0
    - engagement=100, followers=1000, age=48.0 -> (0.1 * 0.5 * 100) = 5.0

    calculate_growth_velocity(current_engagement: int, previous_engagement: int, time_delta_hours: float) -> float
    - Returns (current - previous) / time_delta_hours
    - Returns 0.0 if time_delta_hours == 0
  </behavior>
