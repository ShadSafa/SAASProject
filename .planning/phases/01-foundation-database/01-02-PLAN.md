
3. Create `send_verification_email(email: str, token: str) -> dict`:
   - Construct verification_link: `{settings.FRONTEND_URL}/verify-email?token={token}`
   - Render HTML template (verify_email.html) with context: {verification_link, email}
   - Send via resend_client.emails.send():

     - to: email
     - subject: "Verify your email for Instagram Viral Analyzer"
     - html: rendered template
   - Return {"status": "sent", "message_id": response.id}

   - Construct reset_link: `{settings.FRONTEND_URL}/reset-password?token={token}`
   - Render HTML template (password_reset.html) with context: {reset_link, email}
   - Send via resend_client.emails.send()
   - Same error handling as verification email

5. Add to settings (app/config.py):

   - RESEND_DOMAIN: str (e.g., "yourdomain.com" or Resend sandbox domain)
   - Add to .env.example

```python
from pathlib import Path

def render_template(template_name: str, context: dict) -> str:
    template_path = Path(__file__).parent.parent / "templates" / template_name
    with open(template_path, "r") as f:
        template = f.read()
    return template.format(**context)
```

```python
html = render_template("verify_email.html", {"verification_link": verification_link})
```
